Yale School of Art's design is one I would not expect. Each element has different colors that is not aesthetically pleasing together, but it separates each element with its function in mind. Yale School of Art inspired me to separate elements with different colors and spacing.

Tilde.town uses two vital colors, purple and black. They utilized ASCII art as the focal point and it inspired me to look into ASCII artand include it in my design. I appreciate how they filled in unused parts of the website with a character and create contrast using space rather than color. The title is a unique touch as it is made out of symbols rather than actual letters.

Exaggeration

Contrast is extremely important in enabling users to differentiate between elements. It helps organize and establish a hierarchy of importance that allows the user to efficiently navigate the design. My goal was to create multiple focal points and establish visual interests that users will be unable to distinguish which elements are vital and which ones are secondary. I made vital elements seem like a secondary element by manipulating size and overloaded the user with color contrast and made certain elements not stand out on the background.
